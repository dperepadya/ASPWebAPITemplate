# ASPWebAPITemplate

ASPWebAPITemplate is a template for quickly developing REST APIs with ASP.NET.
It’s built for managing simple To Do tasks but can easily be extended for other purposes.

## Features

- Controllers for handling requests.
- CQRS Pattern for requests and responses.
- Database Repository for data access.
- Entity Framework Migrations feature for managing data models.
- MS SQL and JSON Databases support.
- JWT Authentication for secure access.
- Health Check endpoint for monitoring the API.
- Swagger for testing.

## Technologies Used

- ASP.NET Core
- Entity Framework Core
- MediatR
- FluentValidation
- JWT (JSON Web Tokens)
- Swagger
- MS SQL Server
